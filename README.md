# 🕰️ Time Traveller

An intuitive tool to find the perfect timezone for your target time.

### 🌟 Key Features
- **Smart Tabs**: Switch between `Yesterday` and `Today` logic.
- **Midnight Use-case**: If it's 01:00 AM, the "Tonight" jump correctly points to 11:00 PM of the previous calendar day.
- **Future Travel**: Supports jumping to `Tomorrow` (up to UTC+14).
- **Easter Eggs**: Clicking unavailable times multiple times triggers special feedback (e.g., "Carpe Diem").
- **No Installation**: Runs entirely in your browser.

### 🛠️ Technology
- [Tailwind CSS](https://tailwindcss.com/) for a sleek, dark-mode UI.
- Pure Vanilla JavaScript for timezone offset calculations.
